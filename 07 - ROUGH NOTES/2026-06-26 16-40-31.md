# Obsidian Multi-Device Ecosystem Specification

## 1. Core Architecture
- **Primary Machine**: Laptop (source of truth for templates, workflows, and final merging)
- **Main Sync**: GitHub repository (bidirectional, real-time sync across all devices)
- **Backup**: Kofer WebDAV (passive, scheduled one-way sync — daily or weekly)
- **Devices**:
  - Laptop: Full Obsidian + Git client
  - Phone: Obsidian app + Telegram Bot (for quick capture and light editing)
  - Kobo: KOReader + Custom Git + Editor plugin

All devices sync **directly** with the GitHub repository.

- Templates are dynamically read from Templater folder (keeps bot/plugin in sync with laptop changes)

- GitHub is the single source of truth
- Private folder can be excluded from bot visibility if desired
