# Sad n Sleep

Sad n sleep 



In the corner 
when we weep
moisture in eyes
when we sleep
when we rise
Lies 
enough lies
enough lives
enough cries
sigh
enough sights

lights too bright 
after a while 
too tired 





> *By **[[iink]]***
