# Cloud

Cloud is a sentiment 
It's so fluffy but it's drenched 
To the sorrow that it hold
As it rises and it's dense

through the white
the grey and blacks
shades of tears it holds
a memory stack

Cloud is wanderer
of the skies
It hides the summer 
about happiness; it lies



> *By **[[iink]]***
