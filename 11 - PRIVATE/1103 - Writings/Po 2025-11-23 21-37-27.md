# (Untitled Poem)

When I say I'm not right
I'm not wrong 
Maybe I'm weak
Maybe I'm not strong 


Resentment is a memory
It dies slowly
If i hold it for long
I burn
If don't hold it for long
I turn

Maybe I'll just be a bad memory
